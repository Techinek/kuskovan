Чем могу быть полезен

Консультации по SEO

Консультирую по внутренней оптимизации сайтов: провожу технические аудиты, анализирую структуру сайтов, полноту семантического ядра, корректность настроек счетчиков веб-аналитики и т.д.

Разработка скриптов на Python

Пишу простые скрипты на Python для автоматизации отчетности. Работаю с API различных сервисов: Serpstat, Я.Метрика, Google Analytics, Mailchimp и др.

Веб-аналитика

Настраиваю счетчики веб-аналитики (Google Analytics, Я.Метрика), а также события и цели через Google Tag Manager. Выстраиваю визуализации отчетов в Google Data Studio и Power BI.

Веб-разработка сайтов

Разрабатываю сайты услуг на фреймворке Django 3. В качестве front-end использую Bootstrap 4.
